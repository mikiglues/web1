경로설정

1.절대경로

최상위 디렉토리부터 순차적으로 작성

ex) <img src="c:img.jpg>
<a href="http://abc.com/index.html>첫화면</a>

2.상대경로

사용중인 문서 위치와 사용할 파일 위치에 따라 작성

#1.동일한 디렉토리에 있는경우

파일명.확장자

ex)<img src="logo.jpg">
<a href="notice.html">공지사항</a>

#2.하위 디렉토리에 있는경우

디렉토리/파일명.확장자

ex)<img src="images/main/logo.jpg">
<a href="sub/mypage/emptystate.html">공지사항</a>

#3.상위 디렉토리에 있는경우

../파일명.확장자

../ => 한 단계 위

./ => 최상위 디렉토리

ex) <img src="../logo.jpg"> <img src="../../banner.jpg>
